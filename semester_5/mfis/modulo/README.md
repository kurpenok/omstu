# Modular arithmetic library

#### Usage

```
# Find greater common divisor
cargo run -- gcd --a <a> --b <b>

# Find inverse element in ring modulo
cargo run -- invert --a <a> --m <m>

# Solving simple comparison by modulo
cargo run -- solve --a <a> --b <b> --m <m>

# Solving comparison system by modulo
cargo run -- solve --a <a> --b <b> --c <c> --d <d> --m <m>
```
